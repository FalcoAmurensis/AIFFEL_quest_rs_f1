# 🚀 AIFFEL Research 18기 학습 저장소

이 레포지토리는 **AIFFEL 리서치 18기** 과정의 모든 기술적 도전과 성장을 기록하는 공간입니다. 기초부터 딥러닝 심화, 그리고 실전 프로젝트까지의 여정을 포함하고 있습니다.

## 📅 커리큘럼 구성
본 과정은 총 10개의 모듈로 구성되어 있으며, 주요 퀘스트는 다음과 같은 구조로 관리됩니다.

| 구분 | 주요 내용 | 폴더명 |
| :--- | :--- | :--- |
| **Main Quest** | 과정의 핵심 마일스톤 프로젝트 (01~05) | `MainQuest/` |
| **Exploration** | 다양한 AI 도메인별 실무 실습 (01~07) | `Exploration/` |
| **Going Deeper** | 리서치 중심의 심화 모델 연구 (01~05) | `GoingDeeper/` |
